# BuhLaw Website Assets

Фон сайта: светло-серый #F7F8FA

## Файлы
- BuhLaw_logo_primary.svg — основной логотип (бирюзовый акцент), прозрачный фон
- BuhLaw_logo_inverse.svg — для тёмных фонов
- BuhLaw_icon.svg — иконка (книга+галочка)
- logo_header.png — PNG для шапки сайта
- logo_footer.png — PNG для футера
- favicon.ico, favicon-256.png — фавикон
- buhlaw_colors.css — CSS-переменные с фирменной палитрой

## Рекомендации
- Минимальная ширина логотипа: 160px
- Отступ вокруг логотипа: ≥ высоты буквы 'B'
- На фоне #F7F8FA используйте BuhLaw_logo_primary.svg
- На тёмном фоне #1F3A93 используйте BuhLaw_logo_inverse.svg
